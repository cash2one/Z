ZFramework 出门在外,居家之必备!

示例代码参考Sample/Sample.csproj项目中的源代码

ZFramework主要包含的功能列表

  • 配置
  • 数据访问
  • 简单加密函数
  • 异常处理
  • 日志处理
  • 常用功能封装
    • 数学工具(大小比较)
    • IP地址转换，掩码运算
    • 网络工具，用于检查网络情况
    • 拼音工具，用于计算中文字符串的拼音缩写，英文以原样方式返回
    • 反射工具，可以通过调用此方法寻找调用链中的顶层Web方法名称
    • 邮件发送工具，属于第三方代码，用于发送邮件
    • Web工具, 用于查询客户端IP地址，服务器IP地址，服务器名称，检索Request参数值
    • XmlTools，用于将对象序列化为XML或者反之
    • StringTools，字符串联接功能，字符串拆分，将指定的对象序列化为二进制的BASE64编码字符串
